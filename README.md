# boost
编译boost 到Mac,linux,win32,android,ios

## 编译方式

##### 编译依赖

1、boost v1.66.0 下载地址 :http://www.boost.org/users/download/

##### 当前编译的的库

1、asio 和asio的依赖库

##### [编译]

* macosx
** sh build.sh -macos  	#编译生成boost mac版本的库
** sh build.sh -ios		#编译生成boost ios版本的库
** sh build.sh -android	#编译生成boost android版本的库
